OpenSlides Helm Charts

Production-ready Helm charts for deploying OpenSlides on Kubernetes

Quick Start

Add this Helm repository to your Helm installation:

helm repo add openslides https://jonaskern-dev.github.io/openslides-helm
helm repo update

Available Charts

openslides-k3s-traefik (v1.0.2)
Production-ready chart for k3s/k3d clusters with native Traefik integration

Install the chart:

helm install openslides openslides/openslides-k3s-traefik

Features

  • All 17 OpenSlides microservices
  • PostgreSQL StatefulSet with persistent storage
  • Redis for caching and session management
  • Native Traefik IngressRoutes for k3s/k3d
  • Automatic secret generation
  • Optional cert-manager integration for TLS
  • Multi-environment support (dev/staging/prod)
